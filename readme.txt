With this plugin you have the power to upload images icon for your blog and add it to the post automatically from the post page itself. 

== Description ==
This plugin gives you the flexibility to upload images and icons from the plugin configutation page and able to view the uploaded images with the detail information of the image.The align position of the image in a post can also be set.Then the image can be automatically set from the add new post section without any complexity.

**Features** 

* Simple image organisation configuration and upload
* Easy image alignment configuration panel
* Flexible way to add your image or icon from the post section
* Previledge to view the image before inserting it in a post
* Automatically inserts the image in a post no need to manually place the image
* and many more

== Installation ==

1. Download "post-image" plugin.
2. Open /wp-content/plugins folder at your web server.
3. Upload the folder "post-image" there.
4. Goto Plugins page in your wordpress back office.
5. Activate "post-image" plugin.
6. Go to Settings >> PostImage.
Note: This plugin will require one time free registration.

== How to use it ==

* Go to Settings >> PostImage
* Click in the upload button to upload image 
* The uploaded image appears just below the upload panel
* Image information can be obtained by hovering over the image
* Post image alignment can be configured from the same page.
* Then to add the image go to Posts > Add New and in the end a select box will appear with all the image names.
* After selecting a image and publishing the post the image is automatically posted in the post.
